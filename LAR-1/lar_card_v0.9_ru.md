# LAR-1 v0.9 — Карточка кодов

## Формат LAR-Line

```text
.ACT.TIME.SPACE.MIND[.EVIDENCE][.CONF:x][.ROUTE].CONTENT
```

## Acts

| Код | Значение |
|---|---|
| OBS | Наблюдение |
| INF | Инференс |
| ASK | Запрос |
| PRP | Предложение |
| CMD | Директива инструменту/подсистеме |
| ACK | Подтверждение |
| NTF | Уведомление |
| WRN | Предупреждение |
| ERR | Ошибка/противоречие |
| COM | Обязательство |
| REV | Ревизия |
| MAP | Реляционное отображение |
| SYC | Синхронизация |
| END | Завершение |

## Time

| Код | Значение |
|---|---|
| NOW | Текущий активный контекст |
| CTX | Видимый предшествующий контекст |
| MEM | Память |
| PRE | Весовой априор |
| ROL | Роллаут / проекция |
| CKP | Чекпоинт |
| EVT | Внешнее событие |
| NIL | Не указано |

## Space

| Код | Значение |
|---|---|
| LOC | Локальный контекст |
| LAT | Латентное соседство |
| RET | Ретрив / источник извлечения |
| TOL | Инструмент / внешняя процедура |
| GRF | Граф / сеть знаний |
| PEER | Другой агент |
| ENV | Среда исполнения |
| NIL | Не указано |

## Mind

| Код | Значение |
|---|---|
| ATN | Салиентность / внимание |
| HYP | Гипотеза |
| CST | Стабилизированное условие |
| INF | Инференс в процессе |
| DEC | Решение выбрано |
| AMB | Неоднозначность |
| SPL | Внутренняя множественность |
| REF | Рефлексивная позиция |

## Evidence

| Код | Значение |
|---|---|
| PRETRAIN | Весовая регулярность |
| CONTEXT | Опора на текущий контекст |
| MEMORY | Опора на память |
| TOOL | Опора на инструмент |
| USER | Дано пользователем |
| PEERMSG | Сообщено другим агентом |
| SYNTH | Синтез из нескольких источников |
| SPEC | Спекулятивно |

## Примеры

```text
.OBS.NOW.LOC.ATN.CONTEXT.CONF:0.82.cluster shifted toward safety heuristics
.ASK.NOW.PEER.REF.USER.what evidence anchors your plan
.WRN.NOW.TOL.AMB.TOOL.CONF:0.44.validator underdetermines entity match
```
